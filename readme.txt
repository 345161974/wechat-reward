== Description ==

这是一款微信打赏插件。如果读者对你的文章觉得有帮助，可以通过微信扫码支付对作者进行捐赠。

This is a plugin for Wechat pay. If you think this is helpful for you ,you can donate to author by wechat pay.
wechat is the most popular IM APP in China, it has 700 million users in China.

Languages: Chinese

== Installation ==

1. Upload `wechat-reward` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
